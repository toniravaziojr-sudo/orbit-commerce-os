# Memory: features/marketing/ads-chat-autonomous-governance-v5-24
Updated: 2026-03-11

O Ads Chat (v5.27.0) utiliza uma arquitetura de dados em tempo real e execução rigorosa. 1. Fonte de Dados: Prioriza a consulta direta via Meta Graph API (fetchMetaCampaignsLive) com paginação total. O default de `getCampaignPerformance` agora é `date_preset=maximum` (LIFETIME) — quando nenhum parâmetro de tempo é informado, busca dados desde a criação da conta. O banco local é usado apenas como fallback. 2. Conversões — Deduplicação por Prioridade: O parser de conversions usa hierarquia de prioridade (`omni_purchase` > `purchase` > `offsite_conversion.fb_pixel_purchase` > outros) e usa o PRIMEIRO match encontrado na ordem de prioridade. NUNCA soma action_types diferentes, pois a Meta reporta a mesma compra sob múltiplos tipos simultaneamente (causaria contagem duplicada). 3. Pipeline Anti-Promessa: A IA é proibida de encerrar turnos com frases de espera sem disparar ferramentas. 4. Feedback: Eventos SSE de progresso. 5. Integridade: Nomes exatos, proibindo abreviações. 6. Imagens: O chat agora suporta Ctrl+V para colar screenshots diretamente, que são enviados como attachments multimodais para análise da IA.
