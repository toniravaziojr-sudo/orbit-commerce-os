# Memory: features/marketing/ads-chat-autonomous-governance-v5-24
Updated: 2026-03-11

O Ads Chat (v5.26.0) utiliza uma arquitetura de dados em tempo real e execução rigorosa. 1. Fonte de Dados: Prioriza a consulta direta via Meta Graph API (fetchMetaCampaignsLive) com paginação total. O default de `getCampaignPerformance` agora é `date_preset=maximum` (LIFETIME) — quando nenhum parâmetro de tempo é informado, busca dados desde a criação da conta. O banco local é usado apenas como fallback. 2. Conversões Expandidas: O parser de conversions agora suporta 8 action_types da Meta (`purchase`, `omni_purchase`, `offsite_conversion.fb_pixel_purchase`, `offsite_conversion.custom.purchase`, `onsite_conversion.purchase`, `onsite_web_purchase`, `onsite_web_app_purchase`, `web_in_store_purchase`) e SOMA todos os matches ao invés de usar apenas o primeiro. 3. Pipeline Anti-Promessa: A IA é proibida de encerrar turnos com frases de espera sem disparar ferramentas. 4. Feedback: Eventos SSE de progresso. 5. Integridade: Nomes exatos, proibindo abreviações. 6. Imagens: O chat agora suporta Ctrl+V para colar screenshots diretamente, que são enviados como attachments multimodais para análise da IA.
