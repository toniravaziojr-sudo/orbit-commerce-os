# Memory: features/marketing/ads-chat-autonomous-governance-v5-24
Updated: 2026-03-11

O Ads Chat (v5.25.0) utiliza uma arquitetura de dados em tempo real e execução rigorosa. 1. Fonte de Dados: Prioriza a consulta direta via Meta Graph API (fetchMetaCampaignsLive) com paginação total. Suporta `date_preset=maximum` para dados LIFETIME (desde o início da conta), com paginação de insights até 15 páginas e retry em rate limits. O banco local é usado apenas como fallback. 2. Pipeline Anti-Promessa: Implementa uma regra estrita contra "promessas vazias" (filler promises); a IA é proibida de encerrar turnos com frases de espera sem disparar as ferramentas correspondentes. 3. Feedback: O sistema envia eventos SSE de progresso (type: progress) mapeados por ferramenta para fornecer feedback visual dinâmico no frontend durante operações longas. 4. Integridade: Exige o uso dos nomes exatos das campanhas retornados pelas ferramentas, proibindo abreviações ou invenções. Exige uso de `date_preset: "maximum"` quando o usuário pedir "máximo", "total", "desde o início" ou "lifetime".
